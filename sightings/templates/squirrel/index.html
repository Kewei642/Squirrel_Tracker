Squirrels Tracker

This is a website for keeping track of all the known squirrels and plans to start with Central Park. It includes a map displays the location of the squirrel sightings, a list of all sightings and statistics about sightings. You can view, edit and add the squirrels. The number of squirrels is still growing, and it needs you contribution:)
